# Security Policy

## Supported Versions
Currently, Micro OS is in experimental pre-alpha phase. We are actively accepting security patches for the main `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability
Since we are building a custom kernel and JS runtime, security is critical (especially regarding app sandboxing). 

If you discover a security vulnerability, please **do not** open a public issue. Instead, send a direct message or email to the repository owner. 

We will acknowledge your report within 48 hours and work with you to apply a patch before public disclosure.
